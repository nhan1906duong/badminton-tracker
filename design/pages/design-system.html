Buttons

No thick borders. Soft radius (4px). Accent button = vermilion, used once per screen. Ghost for tertiary actions.

Floating Action Button

Hanko-style square stamp (56×56, 8px radius) anchored bottom-right within a screen container. One per screen, reserved for the screen’s primary creation action (e.g. + New Session). Vermilion accent with a tinted shadow so it lifts off the warm-white canvas without breaking the “hairline, no-shadow” rule for surface cards.

Default · 56×56
Extended · with label
Rules of use
  • One FAB per screen — reserved for the primary creation action.
  • Pin inside the .screen container, not the viewport, so it tracks the phone frame on desktop.
  • Respect env(safe-area-inset-bottom) and add ~104px scroll padding so the last list item clears it.
  • Fade + scale down to 0.9 when a bottom sheet or modal opens so it doesn’t compete with the backdrop.
  • Stamp-in on first paint, lift on hover, scale 0.96 on press. Honor prefers-reduced-motion.
  • Use the extended variant only when the icon alone is ambiguous (e.g. multiple create surfaces).
